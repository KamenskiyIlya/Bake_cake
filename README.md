# Сервис BakeCake

Данный скрипт представлен для автоматизации работы Telegram-бота сервиса BakeCake - магазина тортов на заказ. Бот позволяет клиентам просматривать каталог стандартных тортов, собирать индивидуальные заказы, оформлять заявки и получать информацию о доставке.

## Возможности бота:
- Просмотр каталога стандартных тортов с фото и ценами
- Конструктор торта (выбор коржей, кремов, декора, надписи)
- Расчёт стоимости заказа в реальном времени
- Оформление заявки с контактами и адресом доставки
- Админ-панель для просмотра заявок

## Как установить
Операционная система: любая, где доступен Python 3 (Windows, macOS, Linux). Используем 'pip' для установки зависимостей (aiogram, python-decouple, qrcode, Pillow):

```
pip install -r requirements.txt
```

Рекомендуется использовать virtualenv/env для изоляции проекта.

Примеры запуска программы:


```
python main.py
```

## Переменные окружения
Программа использует нестандартные переменные окружения для настройки. Эти переменные не могут быть автоматически обнаружены и должны быть указаны в файле `.env` перед запуском программы.

TELEGRAM_TOKEN - токен для Telegram-бота (например, 9627688191:AAGRXFdrCSVag0ECkixtjOf1BtvjagFlbMn).

ADMIN_CHAT_ID - Telegram ID для админ-панели (например, 275982955)

PAYMENT_TOKEN - тестовый токен для покдлючения оплаты (например, 9744374395:TEST:1b07130cb8580bb20cf6)
